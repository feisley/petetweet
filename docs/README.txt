c) Extra Features:
	Password is hashed, salted and sent using https (SSL)
	A user can change / update his or her personal profile
	Very nice UI
	Password validation / confirmation upon registering a new account
	An About Page (Shows version number, creators, and date of 			creation along with a nice 'powered by Google Apps' logo).
	Users can view who they are following and who is following them		under their profile as well as all their own tweets.
	Users can send private messages to just one other user using the 		Messaging tab (and see their inbox as well as private 		messages they have sent.)
	Users can reply to messages by clicking reply.
